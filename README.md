A location tracking website and a app.
the app is used to login the user and start tracking service.
the website is a admin panel where you can manage and create users/drivers for mobile app.
admin can view the routes the mobile users have travelled and manage their profile.
the admin can view the routes travelled by the users,the routes are mapped to nearest roads using osrm matching service.

tech stack:
website: react,leaflet
app: react native expo,react-native-maps.
